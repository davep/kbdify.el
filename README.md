# kbdify.el

`kbdify.el` provides a simple command (`kbdify`) that marks up a key
sequence with `<kbd>` tags. So given something like:

```
C-M-S-s-<up>
```

the result will be:

```
<kbd>C</kbd>-<kbd>M</kbd>-<kbd>S</kbd>-<kbd>s</kbd>-<kbd>&lt;up&gt;</kbd>
```

which will result in
<kbd>C</kbd>-<kbd>M</kbd>-<kbd>S</kbd>-<kbd>s</kbd>-<kbd>&lt;up&gt;</kbd>.
